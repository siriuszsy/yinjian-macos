# 音键 1.0.2 使用手册

最后更新：2026-04-21

## 1. 这是什么

音键是一个 macOS 菜单栏语音输入工具。

默认交互很简单：

- 按住 `Fn` 说话，松开后转成文字
- 按住 `Fn + Control` 说话，松开后直接翻译
- `Fn + Shift` 可以在设置里作为备选热键

如果已经授予辅助功能权限，音键会优先把结果直接写回当前光标位置。  
如果还没授权，音键会自动回退到剪贴板，你可以自己粘贴。

## 2. 下载与安装

1. 打开 [GitHub Releases](https://github.com/siriuszsy/voiceKey/releases)。
2. 下载 `voiceKey-1.0.2-macos.zip`。
3. 解压后把 `voiceKey.app` 拖到 `Applications`。
4. 首次启动时，如果 macOS 提示应用来自互联网或开发者未验证，先右键 `voiceKey.app`，选择“打开”。

说明：

- 当前 GitHub Release 版优先保证可用，不等同于 Mac App Store 版。
- 如果系统拦截启动，通常按一次“仍要打开”即可。

## 3. 首次启动后要做的事

### 3.1 麦克风

麦克风权限是必须的。

第一次开始录音时，macOS 会弹出授权框。请允许。  
如果之前点过拒绝，手动去：

- `系统设置 > 隐私与安全性 > 麦克风`

把 `音键` 打开。

### 3.2 辅助功能

如果你希望结果直接落到当前输入框，辅助功能权限也需要打开。

路径：

- `系统设置 > 隐私与安全性 > 辅助功能`

把 `音键` 打开后，建议退出并重新打开一次 app。

如果这项权限没开，音键仍然可以录音、识别和翻译，只是会回退到剪贴板。

### 3.3 键盘监听

默认情况下，`Fn` 热键会先直接尝试注册，很多机器上不需要你手动处理。

只有当你按 `Fn` 或 `Fn + Control` 完全没反应时，再去这里看：

- `系统设置 > 隐私与安全性 > 键盘监听`

如果列表里有 `音键`，把它打开。  
如果列表里没有，先重启 app 再试一次。

## 4. 默认用法

### 4.1 普通听写

1. 把光标放到你要输入文字的位置。
2. 按住 `Fn`。
3. 开始说话。
4. 松开 `Fn`。
5. 音键会先转写，再做基础整理，然后输出到当前输入位置。

### 4.2 翻译

1. 把光标放到目标输入框。
2. 按住 `Fn + Control`。
3. 开始说话。
4. 松开按键。
5. 音键会把语音先识别成文本，再翻译成目标语言。

补充：

- 你也可以先按住 `Fn` 开始说话，如果这句临时想改成翻译，再补按 `Control`。
- 默认翻译目标语言是 `English`，可以在设置里改成 `Chinese`、`Japanese` 或其他语言名/语言代码。

### 4.3 菜单栏入口

菜单栏里有几个常用入口：

- 打开设置
- 写入测试文本
- 退出

如果你怀疑是权限或落字链路有问题，先点一次 `写入测试文本`，不要一上来就跑完整语音链路。

## 5. 如何申请通义千问 / 阿里云百炼 API Key

音键当前使用的是阿里云百炼接口，不是把 API key 内置在应用里。  
所以你第一次用之前，需要自己申请一个百炼 API key。

### 5.1 先准备阿里云账号

如果你还没有阿里云账号，先注册并登录。  
如果控制台提示你还没实名认证，先完成实名认证。

阿里云官方文档：

- 获取 API Key：https://help.aliyun.com/zh/model-studio/get-api-key

### 5.2 开通阿里云百炼

1. 打开阿里云百炼控制台：https://bailian.console.aliyun.com
2. 如果页面顶部提示“立即开通”或类似提示，按页面引导开通。
3. 如果开通过程中提示实名认证，先完成实名认证再回来。

阿里云官方说明里写得很明确：第一次使用前，需要先开通百炼；如果页面顶部没有开通提示，说明你已经开通过。  
参考：

- 获取 API Key：https://help.aliyun.com/zh/model-studio/get-api-key
- 新人免费额度：https://help.aliyun.com/zh/model-studio/new-free-quota

### 5.3 选择地域

音键当前代码固定使用的是北京节点：

- `https://dashscope.aliyuncs.com/compatible-mode/v1`

所以申请 API key 时，建议你在百炼控制台右上角明确切到：

- `华北 2（北京）`

不要先用新加坡或美国地域，否则你后面拿到的 key 和当前 app 的接口地址不匹配。

### 5.4 创建 API Key

1. 在百炼控制台右上角确认地域已经是 `华北 2（北京）`。
2. 进入 `API Key` 页面。
3. 点击 `创建 API Key`。
4. 在弹窗里建议这样选：
   - `归属业务空间`：`默认业务空间`
   - `权限`：`全部`
5. 创建后，点击 API Key 右侧的查看或复制图标，把完整 key 复制出来。

给新用户的建议：

- 如果你只是自己本机使用，先选 `默认业务空间 + 全部`，最省事。
- 如果你后面要做团队隔离或 IP 白名单，再去研究子业务空间和自定义权限。

阿里云官方文档：

- 获取 API Key：https://help.aliyun.com/zh/model-studio/get-api-key

### 5.5 控制费用

如果你只是测试，建议同时做两件事：

1. 领新人免费额度
2. 开启“免费额度用完即停”或你自己的消费保护

官方文档：

- 新人免费额度：https://help.aliyun.com/zh/model-studio/new-free-quota

### 5.6 把 API Key 填进音键

1. 打开音键设置。
2. 找到 `百炼 API Key`。
3. 把刚刚复制的 `sk-...` 粘进去。
4. 点击 `保存 API Key`。
5. 再点一次 `刷新状态`，确认状态变成“已保存”。

当前 app 会优先按这个顺序找 key：

1. 环境变量 `DASHSCOPE_API_KEY`
2. 系统钥匙串里的本地保存项
3. 本地回退文件

对普通用户来说，最简单的方式就是直接在设置页里保存。

## 6. 高级用法：用环境变量配置 API Key

如果你更习惯用环境变量，也可以不在设置页保存，直接配置：

```bash
echo "export DASHSCOPE_API_KEY='你的 sk-xxxxx'" >> ~/.zshrc
source ~/.zshrc
```

然后重新打开音键。

阿里云官方文档：

- 配置环境变量：https://help.aliyun.com/zh/model-studio/configure-api-key-through-environment-variables

## 7. 设置页里常见的几项

### 7.1 听写触发

默认是 `Fn`。

### 7.2 翻译触发

默认是 `Fn + Control`。  
如果你想改，也可以切到 `Fn + Shift`。

### 7.3 目标语言

默认是 `English`。  
你可以改成：

- `Chinese`
- `Japanese`
- `Korean`
- `en`
- `zh`
- `ja`

源语言默认是 `auto`，一般不需要动。

### 7.4 文本整理

默认开启。  
开启后，音键会把口头语、重复词和明显杂音做一次基础整理，让输出更像手打文本。

## 8. 常见问题

### 8.1 说完了，但没有自动落到输入框

先检查辅助功能权限。

- `系统设置 > 隐私与安全性 > 辅助功能`

如果这项没开，音键会回退到剪贴板，这是当前设计，不是故障。

### 8.2 按 `Fn` 没反应

按这个顺序排查：

1. 先确认 app 正在运行
2. 先试菜单栏里的 `写入测试文本`
3. 再看 `系统设置 > 隐私与安全性 > 键盘监听`
4. 打开权限后，退出并重开音键

### 8.3 API Key 保存了，但仍然提示未配置

通常是下面几种情况：

- 粘进去的不是完整的 `sk-...`
- 复制时多了空格或换行
- 你创建的是别的地域的 key
- 账号还没有对应模型调用权限

当前音键默认使用北京节点，所以优先检查你是不是在 `华北 2（北京）` 创建的 key。

### 8.4 能转写，但翻译报错

先检查：

- API key 是否有效
- 百炼账号是否已经开通
- 目标语言是否填写为空或异常

如果只是想确认基础链路没问题，先切回普通听写测试。
